# AI Study Assistant

## Overview
AI Study Assistant is a MERN Stack application that helps students organize notes and generate AI-powered summaries, quizzes, and flashcards from their study material.

## Features

- User Authentication (JWT)
- Create, Edit and Delete Notes
- Upload PDF Notes
- AI Summary Generation
- AI Quiz Generation
- AI Flashcard Generation
- Dashboard
- Responsive UI

## Tech Stack

Frontend
- React.js
- Vite
- React Router
- Axios

Backend
- Node.js
- Express.js
- MongoDB Atlas
- JWT Authentication
- Multer

AI
- Google Gemini API

Deployment
- Frontend: Vercel
- Backend: Render
- Database: MongoDB Atlas

## Installation
